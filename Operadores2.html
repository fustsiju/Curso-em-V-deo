Operadores relacionais, lógicos e ternário.

Operadores relacionais

  • Maior que: ">"
  • Menos quer: "<"
  • Maior igual: ">="
  • Menor igual: "<="
  • Igualdade: "=="
  • Diferente de: "!="

Exemplos em Javascript

  • 5 > 3 ---> True
  • 5 < 3 ---> False
  • 8 >= 8 ---> True
  • 9 <= 7 ---> False
  • 5 == 5 ---> True
  • 4 != 4 ---> False

Operadores lógicos Javascript

  • Negação(Não): "!"
    • !false ---> true
  • Conjunção(E): "&&"
    • true && false ---> false
  • Disjunção(Ou): "||"
    • false || true ---> true

Operador ternário Javascript

O operador ternário tem esse nome pois é composto por 3 partes

Para usar basta seguir esta expressão: Teste?true:false

Um exemplo simples:

Média>=7?"Aprovado":"Reprovado"
